# 英诺绿能冷库智能化设计系统

## 🚀 项目简介
基于 Streamlit 的冷库智能化设计系统，支持冷库参数配置、热负荷计算、设备选型等功能。

## 📋 功能特性
- ✅ 冷库参数配置界面
- ✅ 智能热负荷计算
- ✅ 冷风机智能选型
- ✅ 复叠系统压缩机选型
- ✅ 项目历史管理
- ✅ 数据导出功能
